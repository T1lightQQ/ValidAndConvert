Цель проекта:
Целью проекта является создать библиотеку валидитора типов данных и конвертора в нужные типы данных для использования в других проектах.
В какие данные будет конвертер:
1. to int
2. to Short
3. to Double
4. to String (с int, с double, с char)
Валидатор каких данных:
1. is Int
2. is Short
3. is Long\Long Long
4. is Double
5. is String
6. isInstanceIf
